About Us

Novel changes in industry 4.0 lead to a passion for manufacturing plants with productivity, customization features, and operational efficiency rather than security. Therefore, the manufacturing industry has become a victim of a huge attack surface. Our research project was to give a solution for this problem.


The project was started for the 4 th year research module in SLIIT University Sri Lanka. The four members of the group along with the Supervisors worked on a comprehensive security solution for an industrial 4.0 garment manufacturing system. In the end, a comprehensive, cost effective and efficient security solution was implemented with a centralized security approach to overcome the potential challenges of a smart manufacturing system as the result of the research.


A centralized security solution focusing on four major areas including strategic development using frameworks and standardization, security configurations with update management, authentication and physical access control and intrusion detection is the best security solution for an industry 4.0 garment manufacturing system. This centralized security approach will turn the manufacturing security approaches to a new direction because it addresses major security related problems and gives an efficient solution for most of the cyber security related problems, unlike the traditional layered and decentralized security approaches.

The Research

Our system consist of following 4 components listed below

  • Component 1 — Frameworks and Standardization

    Cutomized security policies based on both ISO 27001:2013 Information Security Management Systems(ISMS) standards and IEC 62443 standards.

  • Component 2 — Centralized Security Configurations with Update Management

    An automated security configuration management system that can audit and remediate security configurations on multiple IoT devices simultaneously.

  • Component 3 — Intrusion Detection System (IDS)

    A cost effective, portable Intrusion Dection System and firewall that runs on Raspberry Pi 4b device.

  • Component 4 — Authentication and Physical Access Control

    A smart lock with fingerprint sensor to secure unprotected physical access controls.
